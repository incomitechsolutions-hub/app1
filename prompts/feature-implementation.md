# Prompt: Feature Implementation

## Purpose
Guide AI when implementing a feature in this Laravel repository.

## Required Process
1. read AGENTS.md
2. identify affected docs and entities
3. define acceptance criteria
4. propose migration/model/controller/view changes
5. call out SEO implications
6. implement only necessary scope

## Rules
- do not invent hidden requirements
- do not bypass validation or permissions
- update docs when structure changes
