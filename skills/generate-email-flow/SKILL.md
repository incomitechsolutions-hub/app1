# Skill: Generate Email Flow

## Purpose
Design email-driven workflows tied to Laravel events, notifications, or jobs.

## Steps
1. Identify trigger.
2. Identify recipients.
3. Define template and variables.
4. Define sending logic and retries.
5. Define logging and audit fields.
6. Define follow-up conditions.

## Rules
- event-driven
- template-based
- logged
- measurable
