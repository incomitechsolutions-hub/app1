# Skill: Implement Feature

## Purpose
Implement a feature in a structured Laravel-aligned way.

## Steps
1. Read `AGENTS.md`.
2. Read relevant docs in `/docs`.
3. Identify affected entities, routes, controllers, services, views, and policies.
4. Define acceptance criteria.
5. Implement migrations/models if needed.
6. Implement validation and authorization.
7. Implement controller/service/view changes.
8. Verify SEO and workflow impact.
9. Summarize changed files and decisions.

## Rules
- follow architecture docs
- enforce business rules
- no direct DB hacks
- no hidden schema changes
