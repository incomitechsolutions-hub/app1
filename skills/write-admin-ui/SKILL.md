# Skill: Write Admin UI

## Purpose
Build admin interfaces inside Laravel using Blade and Tailwind.

## Steps
1. Map the user flow.
2. Group fields clearly.
3. Show workflow state and validation.
4. Support preview and publish actions.
5. Integrate authorization.
6. Keep editor experience practical and fast.

## Rules
- Tailwind only
- workflow first
- clarity over decoration
- backend remains source of truth
